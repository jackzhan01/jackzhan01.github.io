# files/

Put downloadable assets (CV / resume, slides, posters) here.

The homepage links to your CV at `/files/Wangjia_Zhan_CV.pdf`.
To make the CV link work, place your resume PDF in this folder named
exactly:

    Wangjia_Zhan_CV.pdf

If you use a different filename, update the path in:
- `_config.yml`            (author.cv)
- `_data/navigation.yml`   (the "CV" item)
- `_pages/about.md`        (the Contact line)
